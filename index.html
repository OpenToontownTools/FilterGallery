How To Install Filters

  • Navigate to the gallery and find a filter you like
  • Press Download
  • Open the .ZIP file in the program of your choice (i.e. winrar, 7-zip, or Windows Explorer)
  • Navigate to the install directory of Toontown Offline (This can be found by pressing the Directory button on the launcher)
  • Enter the 'filters' folder (or create one if there is not one already)
  • Copy all of the files from inside the .ZIP file you downloaded into the filters folder
